This app is requesting Foursquare API data to show nearest venues.



Location field is required and should be at least 3 characters.
ex.: "Shoreditch, London"

Query field is optional.

If it's not provided, app will use /explore endpoint.

If it's provided, app will use /search endpoint.

ex.: "Pizza", "Donuts"

Search results are diplaying venues names, rating(explore endpoint only) and checkins.
